## 🎨 Personalización

### Cambiar Colores

Edita las variables CSS en `styles.css`:

```css
:root {
    --gold: #FFD700;
    --dark-gold: #D4AF37;
    --rich-gold: #FFA500;
    --deep-black: #0a0a0a;
    /* ... más colores */
}
```

### Cambiar Imágenes

Reemplaza las URLs en `index.html`:

```html
<div class="parallax-bg" style="background-image: url('TU_IMAGEN_AQUI');"></div>
```

### Agregar/Quitar Slides

1. Copia una sección `<section class="slide">` completa
2. Actualiza `data-slide` y el contenido
3. Modifica `const totalSlides` en `script.js`
